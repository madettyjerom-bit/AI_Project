# AI_Project

Actividad Formativa: GitHub Copilot

## Autor
Madette Jerome Sainvil

## Descripción
Este proyecto consiste en un sistema simple de recomendación desarrollado en Python utilizando Visual Studio Code y GitHub Copilot.

## Archivo principal
recommendation_system.py

## Funcionamiento
El programa permite ingresar una categoría de productos y muestra recomendaciones relacionadas.

Categorías disponibles:
- tecnologia
- ropa
- hogar

## Herramientas utilizadas
- GitHub
- GitHub Copilot
- Visual Studio Code
- Python
- Git

## Proceso realizado
1. Creación del repositorio en GitHub.
2. Clonación del repositorio localmente.
3. Creación del archivo recommendation_system.py.
4. Desarrollo del código en Visual Studio Code.
5. Uso de Git y GitHub para subir el proyecto.
